# MyShop

A modern and responsive e-commerce web application built using **React.js** and **Redux Toolkit**. The application allows users to browse products, add items to the shopping cart, update quantities, remove products, and view the total cart value with real-time state management.

---

## Project Overview

MyShop is a frontend e-commerce application designed to provide a smooth shopping experience. It demonstrates efficient state management using Redux Toolkit while maintaining a clean, responsive, and user-friendly interface.

---

##  Features

-  Browse products in a responsive product grid.
-  View product details including image, title, brand, price, and rating.
-  Add products to the shopping cart.
-  Increase or decrease product quantity.
-  Remove products from the cart.
-  Automatic cart total calculation.
-  Real-time cart updates using Redux Toolkit.
-  Efficient global state management.
-  Clean and modern user interface.
-  Fully responsive design.
-  Fast rendering with React components.
